## Установка

#### 1. Установить IceWM:

``` sudo apt install xorg xinit icewm icewm-common ```

#### 2. Создать файл .Xinitrc

``` sudo nano .Xinitrc ```

И написать там:

``` icewm-session ```

#### 3. Запустить скрипт install.sh

```
sudo chmod +x ~/install.sh
~/install.sh
```
